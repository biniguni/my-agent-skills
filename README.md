# My Agent Skills

skills for Claude, Cursor, Antigravity

## Skills List
### Custom 
- **project-scaffold**: Standard project directory generator based on **skill-creator**

### Base
- **skill-creator**: Tool for creating new skills (from Anthropic)  
- **xlsx**: Spreadsheet manipulation tool (from Anthropic)

## Installation
```bash
npx skills add https://github.com/biniguni/my-agent-skills --skill <skill-name>
npx skills add https://github.com/anthropics/skills --skill <skill-name>
```
